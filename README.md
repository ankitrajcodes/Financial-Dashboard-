# 💰 Finance Dashboard

A modern, interactive finance dashboard built with **React + Vite + Tailwind CSS + Framer Motion + Recharts + Zustand**.

Live demo running at **http://localhost:5173/** 🚀

## ✨ Features

✅ **Dashboard Overview**
- Animated summary cards (Balance, Income, Expenses)
- Balance trend line chart (last 7 days)
- Spending breakdown pie chart
- Glassmorphism design with hover animations

✅ **Role-Based Access Control**
- **Viewer**: Read-only access
- **Admin**: Add/edit/delete transactions (+ modal form)
- Role switcher with smooth transitions

✅ **Transactions Table**
- Search, filter by category/type, sort (date/amount/desc/category)
- Row animations on load/filter
- Responsive design (mobile-friendly)

✅ **Smart Insights**
- Top spending category
- Monthly spending trend
- Average & largest transaction stats
- Animated metric cards

✅ **Premium UX**
- Framer Motion animations (stagger, hover lift, slide-in)
- Gradient glassmorphism cards with backdrop blur
- Responsive across all devices
- State persistence (localStorage)
- Modern color palette (indigo/emerald gradients)

## 🛠 Tech Stack
```
React 18 + Vite (fast HMR)
Tailwind CSS (utility-first)
Framer Motion (60fps animations)
Recharts (interactive charts)
Zustand (lightweight state)
localStorage persistence
```

## 🚀 Quick Start
```bash
cd finance-dashboard
npm install --legacy-peer-deps
npm run dev
```
Open **http://localhost:5173/**
